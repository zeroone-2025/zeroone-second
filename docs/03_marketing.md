## 타겟 분석

### 페르소나

#### 페르소나 1 — "오늘만큼은 나도" 김지수

| 항목 | 내용 |
|------|------|
| 이름 | 김지수 |
| 나이/직업 | 21세 / 서울 소재 대학교 2학년 (지방 출신) |
| 주요 고민 | 자체휴강 결심은 했는데 막상 어디 갈지 결정을 못 함. 카카오맵·인스타 투표 등 시도해도 결국 아무것도 못 함 |
| 현재 해결 방법 | 카카오맵으로 카페 검색 → 포기 / 인스타 스토리 투표 → "집가" 답변 받음 / 그냥 다시 눕기 |
| 우리 서비스를 쓰는 이유 | 결정을 '운명'에 넘겨서 죄책감이 줄어드는 느낌. 변명 생성기를 친구한테 보내는 게 루틴이 됨 |

**문제를 겪는 순간**: 월요일 아침 7시 30분, 1교시 통계학 앞두고 이불 속에서 "오늘 하루만"을 결심했지만 갈 데를 못 골라 30분째 유튜브만 보다 점심이 됨.

**첫 반응**: 카톡 단체방 링크 타고 들어와서 큼지막한 "오늘 안 간다" 버튼에 피식 웃음 → 룰렛 돌려서 "한강 🌊 치맥이 부른다" 나오자 아무 생각 없이 나감.

> "내가 결정한 게 아니라 '운명'이 정해준 거니까 죄책감이 조금 줄어드는 느낌?"

#### 페르소나 2 — "어차피 출석 2개 남았어" 이준혁

| 항목 | 내용 |
|------|------|
| 이름 | 이준혁 |
| 나이/직업 | 23세 / 대학교 4학년 (취준 병행 중) |
| 주요 고민 | 자체휴강 결심은 쉽게 하는데, 혼자라 어디 가야 할지 막막함. 친구들은 다 수업 들어감 |
| 현재 해결 방법 | 에브리타임 자유게시판 눈팅 / "서울 혼자 갈 만한 곳" 검색하다 포기 / 결국 학교 도서관 가서 유튜브 봄 |
| 우리 서비스를 쓰는 이유 | 어차피 혼자 결정 못 하는 거, 아예 기계한테 맡기는 게 속 편함. 결과 공유하면서 자체휴강 인증 문화가 재밌음 |

**문제를 겪는 순간**: 오전 9시, 전공 수업 출석 체크 직전. 교수가 PPT 올려주는 거 알고 있고 출석도 2개 남아서 "오늘 하나 써도 되지"라고 결심했지만, 친구들은 다 들어가고 혼자 남겨진 채 교문 앞에서 핸드폰만 만지작거림.

**첫 반응**: 에브리타임 자유게시판에서 다른 학생이 올린 글로 발견 → "어? 이거 뭐야" 하고 눌렀더니 "오늘 안 간다" 버튼이 너무 커서 웃음 → 룰렛 돌려서 "PC방 🎮 운명이 당신을 PC방으로 이끕니다" 나오자 그냥 감.

> "내가 가고 싶어서 간 게 아니라 시스템이 보낸 거잖아. 그러니까 괜찮은 거지."

> **TIP**: 엘리베이터 피치는 04_ppt.md의 Solution 슬라이드에 바로 활용할 수 있습니다.

## 비주얼 컨셉

### 핵심 컨셉

**"자체휴강 충동 감지"** — 형사 경광등(빨간 점멸등)이 번쩍이며 사용자의 자체휴강 충동을 포착, 이왕 이렇게 된 거 제대로 가자고 꼬드기는 흐름

### 색상

| 역할 | 색상 | HEX |
|------|------|-----|
| 메인 컬러 | 소화전 빨간색 | `#CC0000` |
| 서브 컬러 | 흰색 | `#FFFFFF` |
| 서브 컬러 | 거의 검정 | `#111111` |
| 포인트 효과 | 빨간색 점멸 애니메이션 (경광등 효과) | — |

> 파란색 없음 — 빨간 단색 계열로만 사용

### 타이포그래피

- **서체**: Pretendard 또는 Noto Sans KR Bold (굵은 고딕 계열)
- **방향**: 가독성 우선, 경고·알림 느낌을 살리되 읽기 편하게

### 분위기 키워드

`긴박함` / `경고` / `어그로` / `유머` / `일탈 유혹`

### 스토리 흐름 (화면 구성 연계)

1. 경광등 번쩍 → "자체휴강 충동 감지됨"
2. 빨간 버튼 "오늘 안 간다" 클릭 유도
3. 룰렛으로 목적지 결정

### 필수 포함 요소

- 빨간 경광등 아이콘 (점멸 애니메이션)
- 경고 텍스트 스타일 헤드라인
- 서비스 링크 / QR코드 (카드뉴스·영상 마지막)

> **AI 프롬프트** -- 비주얼 컨셉 & 컬러 팔레트 설계
>
> ```
> @docs/01_plan.md 참고해서 비주얼 컨셉을 제안해줘.
> 톤앤매너: {{원하는 느낌}}
> 슬로건: {{선택한 슬로건}}
> 메인/서브 컬러(HEX), 분위기 키워드, 레이아웃 컨셉, 디자인 레퍼런스 3개를 포함해줘.
> ```
>
> **TIP**: 이미지 생성 AI (DALL-E, Midjourney 등)에게 포스터를 만들어 달라고 할 때, 여기서 정의한 컬러/분위기를 프롬프트에 포함하세요.
