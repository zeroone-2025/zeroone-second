# 📋 기획 문서

> **기획자는 팀 리더입니다.** 이 문서의 내용을 최종 결정하는 것은 기획자이며, 여기서 확정된 방향이 개발(`02_dev.md`), 마케팅(`03_marketing.md`), 발표(`04_ppt.md`)의 기준이 됩니다.

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 자체휴강 어디가지? |
| 한줄 소개 | 자체휴강을 결심한 당신에게 오늘 갈 곳을 랜덤으로 정해주는 무책임한 서비스 |
| Pain Point | 수업/출근을 안 가기로 했는데 막상 어디 갈지 모르겠는 결정장애 |

## 컨셉 & 톤앤매너

- **"세상에서 가장 쓸데없는 서비스"**
- 유머러스 & 자조적
- "이 서비스를 사용한 결과에 대해 책임지지 않습니다"
- "부모님께는 비밀로 해주세요"

## 타겟 사용자

- **주요 타겟**: 대학생( 이날에 꼭 자체휴강 어디가야하는 사람.)
- **사용자 특성**: 가끔 일탈을 꿈꾸는 성실한 사람들, 결정장애
- **시장 규모**: 오늘 자체휴강 하고 싶은 모든 사람 (무한대)

## MVP 범위

> 2시간 내에 구현 가능한 핵심 기능만 선정

### Must-have (필수)
- [x] **자체휴강 결심 버튼**: 큰 빨간 버튼 "오늘 안 간다" 클릭 시 화려한 애니메이션
- [x] **랜덤 목적지 룰렛**: 카테고리(카페, 영화관, PC방, 찜질방, 한강 등)에서 랜덤 선택
- [x] **결과 화면**: 선택된 장소 + 축하 메시지

### Nice-to-have (있으면 좋음)
- [ ] **자체휴강 타임라인 추천**: 시간대별 하루 일정 코스 추천
- [ ] **변명 생성기**: 자체휴강 변명 자동 생성 ("지하철 고장", "배탈" 등)
- [ ] **오늘의 통계**: "오늘 47,382명이 함께 자체휴강했습니다" (가짜 통계)
- [ ] **공유 기능**: SNS 공유 버튼

### Out of Scope (이번엔 안 함)
- 실제 장소 추천 (지도 API 연동)
- 사용자 로그인/기록 저장
- 실제 통계 수집

## 화면 구성

| 페이지 | 설명 | 예상 시간 |
|--------|------|-----------|
| 메인 (`/`) | 큰 "안 간다" 버튼 + 오늘의 통계 | 20분 |
| 룰렛 (`/roulette`) | 목적지 뽑기 룰렛 애니메이션 | 30분 |
| 결과 (`/result`) | 선택된 장소 + 변명 + 공유 버튼 | 20분 |
| 타임라인 (`/timeline`) | 자체휴강 하루 일정 타임라인 추천 | Nice-to-have |

## 비즈니스 모델

### 수익 모델

| 단계 | 수익원 | 예상 수익 | 구현 난이도 |
|------|--------|-----------|-------------|
| **MVP** | 제휴 광고 (카페, PC방, 영화관) | 클릭당 50~200원 | 낮음 |
| **Phase 1** | 장소 예약 연결 수수료 | 건당 500~2,000원 | 중간 |
| **Phase 2** | 프리미엄 구독 (월 1,900원) | 월 정기 수익 | 중간 |

#### 1. 제휴 광고 (MVP 단계)
- 결과 화면에 **"여기 어때?"** 버튼으로 제휴 업체 연결
- 예: 룰렛 결과가 "카페"면 → 근처 스타벅스/투썸 쿠폰 노출
- 영화관 → CGV/메가박스 예매 링크 (제휴 커미션)
- PC방 → 제휴 PC방 할인권

#### 2. 예약 연결 수수료 (Phase 1)
- 카카오맵/네이버 예약 API 연동
- 사용자가 추천 장소 예약 시 수수료 발생
- 맛집 예약, 찜질방 입장권, 영화 예매 등

#### 3. 프리미엄 구독 "자휴 패스" (Phase 2)
- **광고 제거** + **프리미엄 기능**
- AI 변명 생성기 (ChatGPT 연동, 더 그럴듯한 변명)
- 맞춤 추천 (날씨, 시간대, 선호도 반영)
- 자체휴강 달력 & 통계
- 월 1,900원 (커피 한 잔 값)

#### 4. 굿즈 & 브랜딩 (추후)
- "오늘 안 간다" 스티커, 티셔츠
- 대학 축제/행사 부스 운영

### 차별점
- **진지하게 쓸데없음** → 바이럴 요소
- 죄책감을 유머로 승화 → 공감 마케팅
- Z세대 감성 (밈, 자조적 유머)

### 확장 가능성
- 카카오 지도 연동하여 실제 근처 장소 추천
- 변명 AI 생성기 (더 그럴듯한 변명)
- 자체휴강 달력 (내가 쉰 날 기록)
- **직장인 버전**: "오늘 반차 어디가지?" (B2B 기업 복지 연계)
- **해외 진출**: "Skip Day" 글로벌 버전
